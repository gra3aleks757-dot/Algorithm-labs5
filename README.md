# Лабораторная работа №5

## Тема: Бинарное дерево поиска (BST)

**Вариативная часть №9**: Найти сумму значений во всех узлах

#  Функции

## Класс Node

__init__ — создаёт один узел с числом. У узла есть левый и правый потомок (пока пустые).

## Класс BST

__init__ — создаёт пустое дерево.

**insert** — добавляет новое число. Если дерево пустое, число становится корнем. Если не пустое, идёт влево или вправо в зависимости от того, больше число или меньше.

**search** — ищет число. Возвращает True, если нашёл, и False, если нет.

**inorder** — обходит дерево в порядке возрастания. Возвращает список всех чисел от меньшего к большему.

**sum_values** — считает сумму всех чисел в дереве. Складывает значение каждого узла.

**_insert_recursive** — вспомогательная функция для вставки. Рекурсивно ищет место для нового числа.

**_search_recursive** — вспомогательная функция для поиска. Рекурсивно идёт по дереву в поисках числа.

**_inorder_recursive** — вспомогательная функция для обхода. Рекурсивно обходит дерево.

**_sum_recursive** — вспомогательная функция для подсчёта суммы. Рекурсивно складывает все значения.

Функция main

**main** — показывает меню. Ждёт, что выберет пользователь. Вызывает нужную функцию. Повторяет, пока пользователь не нажмёт 0.
